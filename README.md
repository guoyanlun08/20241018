#

## 确保环境

Node.js 18

## 开始

构建依赖

```bash
npm i
```

项目启动

```bash
npm run dev
```

[http://localhost:3000/rotate-pdf](http://localhost:3000/rotate-pdf)为页面路径.

## 相关文档

创建和下载 PDFs 文件: [jspdf 文档](https://artskydj.github.io/jsPDF/docs/jsPDF.html)
展示已有 PDFs: [react-pdf 文档](https://github.com/wojtekmaj/react-pdf)
